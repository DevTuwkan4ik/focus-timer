# ⏱️ Focus Timer

A minimalist **Pomodoro timer** for focus, built with vanilla HTML, CSS and JavaScript — no frameworks, no dependencies.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)

## ✨ Features

- Three modes: **Focus** (25 min), **Short break** (5 min), **Long break** (15 min)
- Animated progress ring (SVG)
- Keyboard control: **Space** — start / pause
- Daily counter of completed focus sessions (stored in `localStorage`)
- Browser notifications when a session ends
- Responsive layout, dark theme

## 🚀 Run

No dependencies — just open `index.html` in a browser:

```bash
git clone https://github.com/DevTuwkan4ik/focus-timer.git
cd focus-timer
# open index.html directly, or via a local server:
npx serve .
```
